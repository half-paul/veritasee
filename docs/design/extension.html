§ 5.1 — Browser extension popup · MV3

The overlay follows you to the source.

For readers who'd rather visit the encyclopedia directly, the extension queries our override API for the current page and injects the same correction layer in-place — no proxy round-trip, no ToS posture.

  • VS-040 · Scaffold Manifest V3 for Chrome and Firefox. Background service worker handles auth + API calls; content script injects overlays in-page.
  • VS-041 · Inject Content script queries GET /api/overrides?url=…&anchor=… on page-load and after SPA navigations. Reuses the Reader's correction-panel component, scoped via shadow DOM.
  • VS-042 · Settings Per-domain enable/disable toggle (shown here). Global opt-out. Per-domain ToS notice for sites that have asked us to back off.
  • API latency P95 ≤ 200 ms for the override lookup. Cached client-side keyed by (url, source_revision_hash) for 5 min.
Aesthetic note. The popup carries the same letterpress chrome — circular verity number, vermilion accent, mono metadata — at miniature scale. The browser frame above is paper-on-ink to read instantly as "external context."
